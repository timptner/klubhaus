# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## 2023-06-25

### Added

- Feature to export volunteers for an event as CSV file.
- Display order id on list view to ease reference to payments.
- Position for product images to specify order.

### Changed

- Profile updates which only add new information will be accepted automatically.
- Use separate list for all orders instead of product specific order lists.
- Rework layout of merchandise products to be more appealing.
